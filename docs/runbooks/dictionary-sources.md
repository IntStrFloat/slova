# Источники словаря

Статус: рабочий runbook. Исходники хранятся локально в `.dict-sources/` и не коммитятся; в репозиторий попадают только производные артефакты `assets/dict/*.json` и `assets/levels/*.json`.

## Используемые источники

- `Badestrand/russian-dictionary` — Wiktionary-производные таблицы для существительных, прилагательных и глаголов. Берём **только начальную форму (лемму)** из колонки `bare`; словоформы/склонения из остальных колонок не используются. Чистая морфологическая база без имён собственных, аббревиатур и обрывков.
- `hermitdave/FrequencyWords` `ru_full.txt` — частотный корпус русского языка. Используется только для ранжирования узнаваемости и выделения `ru-common.json`.

## Ожидаемые файлы

```text
.dict-sources/
  nouns.csv
  adjectives.csv
  verbs.csv
  ru_full.txt
```

## Пересборка

```bash
node scripts/build-dictionary.js --src .dict-sources --common-freq 40
node scripts/gen-levels.js
```

Текущий результат:

- `assets/dict/ru.json`: 22448 валидных лемм 3-8 букв.
- `assets/dict/ru-common.json`: 10533 частотных лемм для ответов и базовых слов.
- `assets/levels/world1.json`, `world2.json`, `world3.json`: 250 уровней суммарно (≈5 ответов и ≈4 бонуса на уровень).

## Правила качества

- Рантайм игры не грузит полный словарь; он использует только `answers` и `bonusPool` уровня.
- Слова-ответы берутся из `ru-common.json`, чтобы ранние уровни состояли из узнаваемых слов.
- `bonusPool` берётся из полного `ru.json`, но ограничивается по количеству и сортируется по частотности.
- Обсценная лексика, служебные слова и склеенные формы вроде `чтотo` отфильтровываются в `scripts/dict/stoplists.js` и `scripts/build-dictionary.js`.
