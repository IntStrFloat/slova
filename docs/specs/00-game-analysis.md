# 00 — Анализ оригинала

Статус: утверждено · Обновлено: 2026-06-26

Прототип: «Слова: Игра в слова» (RuStore, издатель SecLife, `com.seclife.rustore.connect.wow.wordscapes.trip`) — RuStore-сборка клона *Words of Wonders / Word Connect*. Категория «Казуальные», рейтинг ~4,7★. Жанр: word-connect + кроссворд.

## Жанровое ядро

- Игроку даётся диск из 3–7 букв.
- Слово составляется **свайпом** по соседним буквам (непрерывное проведение пальца), отпускание = подтверждение.
- Слово засчитывается, если оно есть в целевой **кроссвордной сетке** → буквы появляются в клетках.
- Валидные слова не из сетки («бонусные») не заполняют кроссворд, но дают монеты/наполняют копилку.
- Невалидный набор — отбраковка без штрафа. **Нет жизней/проигрыша** (USP «unlimited play»).
- Кнопка **Shuffle** (бесплатно). Уровень пройден, когда заполнены все клетки.
- Подсказки открывают букву/слово за валюту.

## Прогрессия и мета

- 1000+ уровней, сгруппированы в **локации/страны → достопримечательности** («путешествие по миру»).
- Линейная разблокировка; после блока уровней — карточка достопримечательности в коллекцию.
- Сундуки на карте каждые K уровней.

## Экономика игры

- **Монеты** (soft) — подсказки; **кристаллы** (hard) — премиум-паки.
- Подсказки: «лампочка» (открыть букву), «молоток» (убрать лишние буквы), «открыть слово».
- Копилка (piggy bank) — накопитель монет, разблокировка за IAP.

## Ежедневные/ретеншн-механики

- Логин-календарь (7 дней), колесо фортуны (1 free + rewarded), ежедневный пазл со стриком, push-кампании, спецпредложения (таймерные офферы).

## Соревновательное

- Лидерборды (глобал/неделя/лиги), события/турниры, команды/клубы (в reskin часто отключены — у нас включены, full product).

## Монетизация оригинала и наша модель

- Оригинал: реклама (баннер/интерстишел/rewarded) + IAP (паки валюты, remove-ads, подписка Pro).
- Наша модель: гибрид. Реклама **Yandex Ads** (РФ), платежи **RuStore Billing**. Детали — [05](05-monetization.md), [06](06-economy-currencies.md).

1. Качество русского словаря (валидное слово не засчитано = удар по рейтингу/ретеншну) → [15](15-localization-dictionary.md).

3. Тонкий IAP word-жанра → упор на rewarded + подписку.
