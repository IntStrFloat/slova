## Экраны и состав

### Splash / загрузка
Логотип, прелоад ассетов и шрифтов, init интеграций (ads/iap/push/analytics/backend bootstrap), проверка обновления контент-паков. Затем: первый запуск → `/onboarding`, иначе → `/`.

### Согласие / приватность
Баннер согласия на обработку данных и персонализацию рекламы (152-ФЗ + рекламные сети) до первого показа рекламы.

### Онбординг (туториал, уровни 1–3)
Принудительное обучение: «проведи по буквам» → «собери слово» → «заполни кроссворд». Без проигрыша, с анимацией «руки». В конце soft-ask пушей (см. [12](12-notifications.md)).

### Карта-путешествие `/`
Прокручиваемая карта стран/достопримечательностей; узлы-уровни; индикатор «N/M пройдено»; точки сундуков; иконки входа в мета-экраны; HUD валют (монеты/кристаллы с «+»); бейджи (доступен дейли/колесо/событие). Баннер внизу. → [07].

### Игровой экран `/game`
Зоны: верхняя панель (номер уровня, валюты, пауза); **кроссвордная сетка**; **диск букв** (3–7) со свайпом; кнопка Shuffle; кнопки подсказок (лампочка/молоток/открыть слово); зона бонусных слов. Оверлеи: Pause, LevelComplete, «нет монет». → [03],[06].

### LevelComplete (оверлей)
Анимация заполнения, награда монетами, кнопка «Удвоить» (rewarded), прогресс к достопримечательности, «Далее». Точка `interstitial('level_complete')` по частоте [05].

### Ежедневный пазл `/daily`
Календарь, уровень дня, стрик-награды. → [08].

### Колесо фортуны `/wheel`
Сектора призов, 1 free spin + доп. за rewarded. → [08].

### Коллекция `/collection`
Сетка открыток достопримечательностей (открытые/закрытые), факт по тапу. → [07].

### Магазин `/shop`
Паки монет/кристаллов, `remove_ads`, подписка Pro, активные офферы (таймер). → [05],[06].

### Лидерборд `/leaderboard`
Вкладки: Глобальный / Неделя / Лига. Топ-100 + позиция игрока ±соседи; ник+аватар. → [09].

### События `/events`
Активные турниры, таблица очков события, таймер, награды. → [10].

### Команды `/teams`
Поиск/создание/вступление в клуб, командный прогресс, вклад участников. → [10].

### Профиль `/profile`
Ник+tag, аватар, статистика (уровней/слов/стрик), статус аккаунта/синка, привязка. → [11].

### Настройки `/settings`
Звук/музыка/вибро (тумблеры), язык, уведомления, восстановить покупки, привязка аккаунта, политика/поддержка, версия.

### Сервисные оверлеи
«Нет монет» (купить/видео), Special Offer (таймер), «Нет интернета» (для онлайн-фич; ядро играбельно оффлайн).

## Карта переходов

```
Splash → (Согласие) → [первый запуск: Онбординг] → Карта `/`
Карта → tap узел → `/game` → LevelComplete → (interstitial) → Карта/след. уровень
LevelComplete → «Удвоить» → rewarded → x2 монеты → Карта
Карта → иконки → {`/daily` | `/wheel` | `/collection` | `/shop` | `/leaderboard` | `/events` | `/teams` | `/profile` | `/settings`}
Подсказка → {списание | «нет монет» → `/shop`/rewarded}
N уровней → авто-оверлей открытия достопримечательности → `/collection`
```
