# 06 — Экономика и валюты

Статус: утверждено · Обновлено: 2026-06-26

Все числа — в `core/config/economy.ts`. Балансировка — отдельная задача (телеметрия из [16](16-analytics.md)).

## Валюты

| Валюта | Тип | Источники | Траты |
|---|---|---|---|
| Монеты (soft) | основная | grid/bonus-слова, сундуки, колесо, дейли, копилка, rewarded | подсказки |
| Кристаллы (hard) | премиум | IAP, редкие награды | премиум-подсказки, эксклюзивные паки |

Стор `features/currency` (`currency.v1` = `{ coins, gems }`): `add(kind, n, source)`, `spend(kind, n, sink) → ok|insufficient`. Все мутации логируются в аналитику (source/sink) для баланса.

## Подсказки (`features/hints`)

| Подсказка | Эффект | Стоимость (старт) |
|---|---|---|
| Лампочка | открыть одну букву (случайную или выбранную клетку) | 50 монет |
| Молоток | убрать лишние буквы с диска / открыть короткое слово | 120 монет |
| Открыть слово | заполнить целое слово сетки | 300 монет |

- Альтернатива оплате — **rewarded «бесплатная подсказка»** (флаг [05]).
- Подписка Pro даёт N бесплатных подсказок на уровень (конфиг).
- При нехватке монет → оверлей «нет монет» → Магазин/rewarded.

## Копилка (piggy bank)

Бонусные слова и часть наград капают в копилку (`piggy`); по достижении порога — разблокировка содержимого (монеты) **через IAP-покупку** `unlock_piggy` или по достижении бесплатного потолка (конфиг). Anti-frustration: визуальный прогресс наполнения.

## Магазин (`/shop`)

Секции: Паки монет, Паки кристаллов, `remove_ads`, Подписка Pro, Активные офферы (таймер). Цены тянутся из RuStore Billing (`getProducts`), плейсхолдеры в Noop. Special Offer — стартовый пак новичку + выходные акции (флаги/таймеры).

## Балансировка (целевые ориентиры)

```ts
export const ECONOMY = {
  coins: { perGridWord: 5, perBonusWord: 2, levelClearBonus: 20, chest: [50,120,300] },
  hints: { bulb: 50, hammer: 120, revealWord: 300 },
  wheel: { sectors: [10,20,50,'bulb',100,'gems5','jackpot'] },
  daily: { loginCalendar: [10,15,20,25,30,40,'bulb'], puzzleStreakBonus: 5 },
  piggy: { capFree: 200, unlockPriceGems: 0 },
  pro: { freeHintsPerLevel: 2, dailyMultiplier: 2 },
  starterOfferCoins: 1000,
};
```

Принцип: монеты должны слегка «поджимать» к ~10–15 уровню (чтобы rewarded/IAP имели ценность), но **никогда не блокировать прогресс** (нет жизней). Подсказки — ускоритель, не обязательны.

## Тесты

`features/currency`/`hints`: начисление/списание, insufficient-путь, бесплатные подсказки Pro, лимиты копилки, идемпотентность наград (не задвоить за один источник).
