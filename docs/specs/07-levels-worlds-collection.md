# 07 — Уровни, миры, коллекция

Статус: утверждено · Обновлено: 2026-06-26

## Модель прогрессии

- Уровни линейны (1..N), сгруппированы в **миры** (страны/регионы), миры → **достопримечательности** (блоки уровней).
- Конфиг `core/config/worlds.ts`: `WORLDS = [{ id, title, theme, landmarks: [{ id, title, fact, image, levelRange }], levelCount }]`.
- Стор `features/levels` (`levels.progress`): `{ currentLevel, completed: Set, worldUnlocks }`.
- Разлок: уровень `n+1` доступен после завершения `n`. Мир разблокируется по достижении его первого уровня.
- Текущий v1-контент: глобальная прогрессия 1..250; `world1` = 1..90, `world2` = 91..170, `world3` = 171..250.

## Карта-путешествие (`/`, хаб)

- Вертикально прокручиваемая «дорога» узлов-уровней по мирам; фон/декор — `worldThemes` [04].
- Узлы: пройден / текущий / заблокирован; точки **сундуков** каждые `worldChestEvery` уровней [03/06].
- HUD валют, бейджи доступности мета-фич (дейли/колесо/событие), кнопки входа в мета-экраны [01].
- Тап по доступному узлу → `/game` с `levelId`.

## Загрузка контента уровней

- Паки `assets/levels/<world>.json` (генератор [03]). Рантайм грузит пак нужного мира лениво (expo-asset), не весь контент сразу [13].
- Картинки достопримечательностей — докачка по миру (expo-image cache).
- Fallback: если пак мира отсутствует — мир «скоро» (без краша).

## Сундуки

- На узлах-сундуках: награда монетами (`ECONOMY.coins.chest`), иногда кристаллы/подсказка. Открытие — анимация [04]; rewarded «удвоить сундук» (флаг [05]).

## Коллекция достопримечательностей (`/collection`)

- Стор `features/collection` (`collection.v1`): открытые `landmarkId`.
- После завершения последнего уровня достопримечательности → авто-оверлей reveal (фото + факт) → запись в коллекцию.
- Экран: сетка карточек (открытые цветные / закрытые силуэты), тап → факт. Мета-цель «собрать все».
- Прогресс-бар «N/M открыто» на мир и суммарно.

## Темы миров

- Каждый мир задаёт `worldTheme` (палитра/декор поля). Применяется на `/` и `/game`. Возможен ручной выбор разблокированной темы в Настройках (как в blockblast).
- Фон игрового поля `/game` — арт **текущей достопримечательности-блока** (`landmarkForLevel`: первая с `untilLevel ≥ levelId`), а не один фон на мир: сцена меняется по мере прохождения блоков (Эйфелева → Лувр → Нотр-Дам → Арка). Если арта достопримечательности нет — фолбэк на фото мира. Смена арта — кроссфейд `expo-image`.

## Тесты

`features/levels`: разлок/гейтинг, переход между мирами, идемпотентность завершения; `features/collection`: открытие без дублей, прогресс.
