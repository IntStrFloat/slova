# 08 — Ежедневный пазл, колесо, логин-стрик

Статус: утверждено · Обновлено: 2026-06-26

Все механики — на локальной дате устройства с защитой от перевода часов (см. ниже). Числа — `ECONOMY.daily`/`ECONOMY.wheel` [06].

## Логин-календарь (`features/dailybonus`)

- `daily.bonus` = `{ lastDay, count }`. При первом входе за день — награда по лесенке (день 1..7), затем цикл/повышенная неделя.
- Пропуск дня → сброс на день 1 (конфиг: мягкий сброс/защитник).
- UI: карточка с подсветкой текущего дня, анимация выдачи [04].

## Ежедневный пазл (`/daily`, `features/dailypuzzle`)

- `daily.puzzle` = `{ lastDay, streak, doneToday }`. Один специальный уровень в день (детерминирован по дате через генератор [03], seed = дата).
- Прохождение → стрик +1, бонус `puzzleStreakBonus × множитель`; пропуск → стрик сброшен.
- Календарь месяца: отмеченные дни; награда за серии (7/30).

## Колесо фортуны (`/wheel`, `features/wheel`)

- `wheel.v1` = `{ lastFreeSpinDay, spinsToday }`. 1 бесплатный спин/день + до `maxRewardedSpins` за rewarded [05].
- Сектора из `ECONOMY.wheel.sectors` (монеты/подсказка/кристаллы/джекпот). Результат — серверо-независим (локально), анти-фрод не критичен (награда мелкая).
- Анимация вращения + хаптика остановки [04].

## Защита от перевода времени

- Хранить `lastDay` как `YYYY-MM-DD` локально; дополнительно — last-seen server time при онлайне (через `/api/time` или заголовок). Если устройство «прыгнуло» назад относительно `lastSeenServer` → не выдавать повторно. Простая эвристика без жёсткой блокировки оффлайна.

## Push-связка

Пуш-кампании «новый дейли», «крути колесо», «не теряй стрик» — через [12](12-notifications.md).

## Тесты

Переходы через полночь, сброс/инкремент стрика, лимиты спинов, идемпотентность дневной выдачи, защита от отката времени.
