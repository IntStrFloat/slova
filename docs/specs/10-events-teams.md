# 10 — События/турниры и команды/клубы

Статус: утверждено · Обновлено: 2026-06-26

Онлайн-фичи поверх backend [11](11-profile-cloudsave-backend.md). Опциональны за флагами (`FEATURES.events`, `FEATURES.teams`) — приложение работает без них.

## События/турниры (`features/events`, `/events`)

- Временное соревнование с окном `[startsAt, endsAt]` и отдельной метрикой-очками (например, «слова за выходные», «бонусные слова»).
- Сервер определяет активные события (`GET /api/events`), принимает вклад (`POST /api/events/:id/progress`, ревалидируемый), ведёт таблицу (`GET /api/events/:id/board`) и начисляет награды по завершении.
- UI: список активных событий, таймер, моя позиция/очки, призовые тиры, правила. Бейдж на Карте при активном событии [07].
- Анти-чит: вклад выводится из подтверждённого прогресса, не из произвольного клиентского числа.

## Команды/клубы (`features/teams`, `/teams`)

- Игрок состоит максимум в одном клубе. Действия: поиск/создание (стоимость — кристаллы, конфиг)/вступление (открытый/по заявке)/выход.
- Командный прогресс: суммарный вклад участников в командную цель/событие; общая награда всем участникам при достижении.
- Роли: лидер (kick/настройки), участник. Лимит участников (конфиг).
- Эндпойнты: `/api/teams` (CRUD/поиск), `/api/teams/:id/join|leave`, `/api/teams/:id` (состав+прогресс).
- UI: карточка клуба, список участников с вкладом, прогресс-бар цели, чат — **вне скоупа v1** (модерация/комплаенс), только реакции/эмодзи опционально позже.

## Модель данных (backend)

```
Event { id, type, startsAt, endsAt, rules, rewards }
EventEntry { eventId, userId, score, updatedAt }
Team { id, name, leaderId, members[], emblem, createdAt, joinPolicy }
TeamProgress { teamId, goalId, total, updatedAt }
```

## Деградация

При оффлайне/ошибке — экраны показывают кэш + «нет связи», вклад копится в очередь и досылается. Награды событий/команд начисляются сервером (идемпотентно по `eventId/teamGoalId`).

## Тесты (backend)

Жизненный цикл события (старт/вклад/итог/награды-идемпотентность), вступление/выход из клуба, агрегация вклада, лимиты.
