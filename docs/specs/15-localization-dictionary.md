# 15 — Локализация и словарь

Статус: утверждено · Обновлено: 2026-06-26

Качество русского словаря — критический фактор рейтинга/ретеншна [00]. Любое валидное слово, которое «не засчиталось», — баг.

## i18n (`core/i18n`)

- Мини-i18n без либ: словари `ru`/`en`, `t(key, params)`. Основной язык — русский; en — задел.
- Все UI-строки — через `t`. Числа/даты — Intl/локаль.

## Игровой словарь (контент, не UI)

- Источник: открытый русский морфологический словарь (Wiktionary-производная база Badestrand) + частотный корпус hermitdave/FrequencyWords — **только нарицательные**, без аббревиатур/имён собственных/обсценной лексики.
- Берём **только начальные формы (леммы)** — колонка `bare` (сущ. → им.п. ед.ч., прил. → м.р. ед.ч., глаг. → инфинитив). Склонения/спряжения **не используются**: кроссворд и бонусы оперируют словарными формами, а не косвенными падежами (`людей`, `юле`).
- Пайплайн (`scripts/build-dictionary.js`):
  1. Чтение лемм из источников → нормализация (нижний регистр, `ё→е`, убрать дефисы/пробелы/ударения).
  2. Фильтры: длина 3–8, стоп-лист (служебные слова для `common`, мат/оскорбления для всех словарей).
  3. Дедуп, сортировка, частотные ранги.
  4. Выход:
     - `assets/dict/ru.json` — полный валидационный словарь для бонусных слов;
     - `assets/dict/ru-common.json` — частотный узнаваемый слой для ответов/сидов;
     - `assets/dict/ru.meta.json` — счётчики сборки.
  5. Эти артефакты **используются только генератором/бэкендом**, не грузятся в рантайм игры [03/13].
- Источники и команды пересборки — в `docs/runbooks/dictionary-sources.md`.

## Использование

- Генератор [03] берёт словарь для `answers`/`bonusPool` уровня.
- Бэкенд [11] — ту же выжимку для ревалидации (или принимает определение уровня как источник правды).
- Рантайм-валидация слов идёт по пулу уровня, не по мегасловарю.

## Контроль качества

- Стоп-лист обязателен (комплаенс RuStore/возрастной рейтинг).
- Набор регресс-тестов: «эти слова обязаны приниматься / эти — отвергаться».
- Обратная связь: события аналитики `word_rejected_candidate` (часто отвергаемые игроками строки) → пополнение словаря.

## Тесты

Нормализация (`ё/е`, регистр), фильтры длины/стоп-листа, детерминизм выжимки, регресс-набор валид/невалид.
