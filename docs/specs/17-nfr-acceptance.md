## Критерии приёмки по фазам (вехи)

- **M1 (играбельный срез):** один мир, генератор даёт валидные уровни, свайп/сетка/бонусы/shuffle работают, прогресс/resume сохраняются, движок покрыт тестами, оффлайн.
- **M2 (экономика+монетизация+дейли):** монеты/подсказки/магазин, Yandex Ads (banner/interstitial/rewarded) по частотам, IAP-интерфейс (stub→RuStore), дейли-пазл/колесо/логин-стрик.
- **M3 (мета):** карта-путешествие, миры/темы, коллекция достопримечательностей, онбординг, сундуки.
- **M4 (онлайн-ядро):** backend задеплоен (изолированно), профиль/bootstrap, облачный сейв, ревалидация прогресса, лидерборды + лиги.
- **M5 (соц):** события/турниры, команды/клубы — с серверной идемпотентностью наград.
- **M6 (релиз):** push, аналитика-воронки, перф-полировка, локализация, signed AAB, чек-лист RuStore пройден.

## Definition of Done (любая задача)

Код + тесты зелёные + типы + соответствует спеке раздела + не нарушает анти-чеклист/бюджет зависимостей + (если онлайн) backend-тест + обновлён соответствующий спек-файл при изменении контракта.
