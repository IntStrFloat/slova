# 18 — Целевая аудитория и дизайн под неё

Статус: утверждено · Обновлено: 2026-06-26

## Кто это (30–35+)

Жанр word-connect/кроссворд тяготеет к **взрослой casual-аудитории 30–35+** (ядро 35–55+, заметная доля женщин). Это **не «геймеры»**, а люди, играющие для:
- **расслабления и снятия стресса** (вечер, перерыв, транспорт);
- **«держать ум в тонусе»** (словарь, лёгкий вызов без давления);
- **escape/мечты о путешествиях** — отсюда сильная эмоциональная отдача от **реальных фото достопримечательностей** («виртуальный тур по миру»).

Контекст: РФ/RuStore, средний Android, игра одной рукой, переменное освещение, у части аудитории — возрастное ухудшение ближнего зрения (40+).

## Что это значит для дизайна (приоритеты)

| Принцип | Решение в продукте |
|---|---|
| **Читаемость** | Крупная типографика (body 17, title 22, display 34), высокий контраст ≥4.5:1, тень текста поверх фото, без «серый-на-сером» |
| **Спокойствие/премиальность** | Тёплая сдержанная палитра (word-green + reward-gold), serif-заголовки (Lora) + чистый sans (Nunito); НЕ неон/брутализм/детский тон |
| **Тактильность/ясность** | Claymorphism: крупные скруглённые «clay»-панели с мягкой тенью, очевидные аффордансы, кнопки с press-scale |
| **Крупные тач-таргеты** | CTA ≥62, кнопки ≥52, иконки-кнопки 48, карточки навигации крупнее, увеличенные отступы |
| **Эмоция/путешествие** | **Реальные фото** достопримечательностей вместо абстрактных силуэтов (Эйфелева башня, скайлайн, Кремль) |
| **Навигация** | Предсказуемая, конвенциональная; иконка+подпись (не icon-only); никаких скрытых жест-онли действий |
| **Движение** | Мягкое, уважать reduced-motion; ничего резкого/мигающего |
| **Без фрустрации** | Нет жизней/таймеров-давления; подсказки и rewarded — помощь, не наказание |

## Монетизация под аудиторию

- Аудитория 35+ платит лучше среднего casual и устойчива к **rewarded** (добровольная помощь) и **подписке Pro** (без рекламы + бонусы). Реклама — ненавязчивая, по анти-чеклисту [05](05-monetization.md).
- Реальные фото-локации усиливают вовлечённость в коллекцию → ретеншн → LTV.

## Реализация (v3, отражено в коде)

- `ui/WorldBackground` — реальные фото (`assets/worlds/*.jpg`) + градиентный скрим.
- `ui/theme` — палитра/радиусы/`shadowCard` под clay; `ui/AppText` — Lora/Nunito, крупнее, тень.
- `ui/primitives` — крупные кнопки/панели с тенью; увеличенные тач-таргеты на экранах.
- Производство: фото-ассеты заменить на **курируемые лицензированные** (текущие — Flickr CC через LoremFlickr для демо); следить за весом бандла и докачкой по мирам [13](13-performance.md).
